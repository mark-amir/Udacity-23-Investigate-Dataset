Hello, welcome to our project "Investigate a dataset" and our dataset in this project is about "Movies" ... Yeah , I know that we all enjoy watching movies and everyone has its prefered genre .

Dataset Description¶

Let's take a deep look about this dataset ... This data set contains information about 10,000 movies collected from The Movie Database (TMDb):

1) There are some columns, like cast and genres, contain multiple values separated by pipe (|) characters

2) The final two columns ending with _adj show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

3) The dataset has 10866 rows and 21 columns with total values 228,186 values !!

Let's take another deep look about the columns:
id = represents the unique general identifier of each film

imdb_id = represents the unique identifier of each film of Internet Movie Database (imdb)

popularity = represents the popularity of the film

budget = represents the amount of money is needed for the film's production

revenue = represents the amount of money is that the film earned

original title = represents the original name of the film

homepage = represents the official website pf the film's show

director = represents the name of the director who made the film

tagline = represents a short sentence tht is used to promote the film's show

overview = represents a brief summary about the film

runtime = represents the duration of the film

genres = represents the type of the film

production companies = represents the company that produces the film

released date = represents the specific day, month, year when the film became available to the public

vote count = represents the number of votes of the film

vote average = represents the average rating of the film

release year = represents the year when the movie was released

The final two columns ending with _adj show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

In [27]:
def grouping_data (df, column_1, column_2):
    grouped_data = df.groupby(column_1)[column_2].sum().sort_values(ascending = False)
    return grouped_data

Here we make a function is called grouping_data to split the dataFrame into specified groups and applying opeations on them then combine the results of the chosen columns column_1 and column_2 which we will assign their names from the dataset by using .groupby() method

Also we used another methods like :

.sum() method to sum all values of the chosen columns
.sort_values() method to arrange the values , because of the ascendingly arrangement of this method , we wrote ascending = False between the () so we can know the arrangement descendingly
We assigned all of this to grouped_data variable

Then we use return to return the value of variable
